# Projeto Final - Programação Orientada a Objetos

**● Descrição:**

Este projeto foi desenvolvido para a disciplina **Programação Orientada a Objetos**, na universidade **Universidade Católica de Pernambuco - UNICAP**.

**● Sobre o Projeto:**

Jornada do Desenvolvedor é um jogo que simula a progressão de carreira de um desenvolvedor de software. Gerencie sua energia, ganhe experiência, complete projetos e evolua de Júnior até CEO! O sistema foi desenvolvido para o aprendizado do framework JavaFX.


# Mecânicas de Jogo

▸ Sistema de Progressão: Júnior → Pleno → Sênior → CEO

▸ Gestão de Energia: Energia se regenera automaticamente ao longo do tempo

▸ Sistema Econômico: Ganhe gold trabalhando e gaste na loja

▸ Experiência (XP): Estude e trabalhe para ganhar XP e subir de nível

# Estrutura do Projeto

| Diretório | Descrição |
|-----------|-----------|
| `src/main/java/com/careersim/model/` | Classes de modelo (Desenvolvedor, Projeto, Item, etc) |
| `src/main/java/com/careersim/shop/` | Sistema de loja e itens |
| `src/main/java/com/careersim/ui/` | Interface JavaFX |
